Watched locations

Leave the topic blank for a watch-only location - it's tracked and shown on the map (and widens the capture area) but never sends an ntfy alert, so you can run without ntfy.

Public ntfy topics need no token - leave it blank. For a private topic, enter the name of the environment variable holding the token (e.g. NTFY_HOME) and set that variable in the daemon's environment. Never paste the token itself.

Once the server and topic are filled, Test ntfy unlocks - click it to send this exact server/topic a sample alert and confirm it reaches your device before saving.

Tip: click 📍 Pick on map, then click the map to drop a pin (drag it to fine-tune). The dashed circle previews the alert radius - change Radius to resize it.
